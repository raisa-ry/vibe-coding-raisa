# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project: HikePack AI

AI-powered hiking gear list generator — Spring Boot 3.4 + vanilla JS single-page app.

## Project Setup

- **IDE**: JetBrains IntelliJ IDEA
- **Language**: Java 21 (LTS)
- **Build**: Maven (`pom.xml` at repo root)
- **VCS**: Git (remote: `https://github.com/raisa-ry/vibe-coding-raisa.git`)

## Key Commands

```bash
# Run (stub mode — no API key needed)
"/Applications/IntelliJ IDEA.app/Contents/plugins/maven/lib/maven3/bin/mvn" spring-boot:run

# Run with Claude AI generation
ANTHROPIC_API_KEY=sk-ant-... "/Applications/IntelliJ IDEA.app/Contents/plugins/maven/lib/maven3/bin/mvn" spring-boot:run

# Build fat JAR
"/Applications/IntelliJ IDEA.app/Contents/plugins/maven/lib/maven3/bin/mvn" package -DskipTests

# Run JAR
java -jar target/hikepack-ai-1.0.0.jar
```

**Note:** System `mvn` is not on PATH; use IntelliJ's bundled Maven at the path above, or open in IntelliJ and use the Maven panel.

## Architecture

```
src/main/java/com/hiking/
  HikingPackListApplication.java     # Spring Boot entry point
  controller/PackListController.java # POST /api/generate
  service/PackListService.java       # Claude API call + prompt builder
  model/TripInput.java               # Request body
  model/PackListResponse.java        # Response (categories, what-ifs, tips)
  model/Category.java
  model/WhatIfScenario.java

src/main/resources/static/
  index.html   # Single-page form + results
  style.css    # Earth-tone responsive styles + print media
  app.js       # Form → API → render; share link (URL hash); print
```

## Environment Variables

| Variable | Required | Description |
|---|---|---|
| `ANTHROPIC_API_KEY` | No | Anthropic API key; falls back to logic-based stub if unset |
| `SERVER_PORT` | No | Override port (default 8080) |

## Features

1. **Trip inputs** — duration (hours/days), temp range, weather checkboxes (rain/wind/snow/heat), terrain, experience level
2. **AI pack list** — 5 gear categories generated by `claude-sonnet-4-6`
3. **What-if scenarios** — Cold Snap, Sudden Storm, Injury, Getting Lost (each with extra gear + actions)
4. **Pro tips** — trip-specific safety/comfort advice
5. **Shareable link** — form state base64-encoded in URL hash (`#share=...`); auto-generates on load
6. **Print view** — `@media print` hides form, expands all scenarios, formats for paper
